# AWS-Flask-App

Flask app source code that is hosted on AWS using an EC2 instance

http://ec2-3-150-132-158.us-east-2.compute.amazonaws.com/ (Currently not running)

## Environment Setup

Create `configs/environment.env` file containing values for `FLASK_SECRET_KEY` and `DATABASE`. Example .env file can be found at `configs/environment.env.example`.
